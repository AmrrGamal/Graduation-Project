User Guides

Step-by-step guides for each user role

Doctor's Guide

Complete guide for dental professionals

Getting Started:
  1. Create your account and complete the verification process
  2. Set up your professional profile with your credentials
  3. Configure your notification preferences
Using the System:
  1. Upload patient X-rays through the dashboard
  2. Review AI-generated analysis and recommendations
  3. Add your professional notes and observations
  4. Generate and save detailed reports
  5. Track patient history and progress
Best Practices:
  • Always verify AI recommendations with your professional judgment
  • Keep patient records updated regularly
  • Use the system's tracking features for follow-ups
Student's Guide

Learning resources for dental students

Getting Started:
  1. Register with your educational institution email
  2. Complete the student verification process
  3. Set up your learning profile
Learning Features:
  1. Access the educational case library
  2. Practice with sample X-rays and cases
  3. Review AI analysis and explanations
  4. Take quizzes and assessments
  5. Track your learning progress
Learning Tips:
  • Start with basic cases before moving to complex ones
  • Review the AI explanations carefully
  • Use the practice mode regularly
Admin's Guide

System management and administration

Getting Started:
  1. Access the admin dashboard
  2. Set up system configurations
  3. Configure user roles and permissions
Management Features:
  1. Monitor user registrations and verifications
  2. Manage user accounts and access levels
  3. Review system usage and analytics
  4. Handle support requests and issues
  5. Generate system reports
Administration Tips:
  • Regularly review system logs and activities
  • Keep user permissions up to date
  • Monitor system performance and usage
